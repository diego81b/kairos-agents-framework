3 SIMPLE STEPS
================================================================================

STEP 1: Extract ZIP
  $ unzip kairos-framework-v2.0.zip
  $ cd kairos

STEP 2: Create GitHub Repo
  $ git init
  $ git add .
  $ git commit -m "feat: KAIROS Framework v2.0"
  $ git remote add origin https://github.com/comm-it/kairos.git
  $ git push -u origin main

STEP 3: Deploy - Pick one:

  Vercel (5 min):
    $ npm install -g vercel
    $ vercel --prod

  GitHub Pages (10 min):
    Settings → Pages → Source: main, Folder: /docs

  Netlify (10 min):
    Connect repo on netlify.com
