# FAQ

**Q: Do I need to learn all 7 agents?**

A: No. Just mention "KAIROS" and the orchestrator handles everything.

---

**Q: Which tool should I use?**

A: Claude Code (free start) or Cursor (best IDE integration).

---

**Q: Can I customize agents?**

A: Yes. Edit the agent markdown files in the `agents/` folder to match your team's coding patterns, preferred libraries, or internal standards.

---

**Q: Does it work with existing code?**

A: Yes. Agents analyze your codebase and match its existing patterns when generating new code.

---

**Q: What about legacy projects?**

A: Agents handle any tech stack — Node.js, Python, Java, Go, and more.

---

**Q: Is the output production-ready?**

A: 80-90% ready. Always review before shipping. The Code Reviewer and Test Verifier agents catch most issues, but human oversight is still recommended.

---

**Q: How do I measure ROI?**

A: Track `(hours before - hours after) × hourly rate` per feature. See [Metrics & ROI](./metrics) for detailed numbers.

---

**Q: Can teams collaborate?**

A: Yes. Use Claude Teams or Cursor for multi-developer workflows. Agents and their configurations live in the repo and are shared automatically.
